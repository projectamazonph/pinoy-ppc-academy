# Pinoy PPC Academy Documentation

This directory is the documentation-first source of truth for the public greenfield repository `pinoy-ppc-academy`.

## Product

Pinoy PPC Academy is built for Filipinos who want to start a virtual assistant career or shift into a higher-value specialization. The learning pathway begins with VA career readiness, progresses through Amazon foundations and PPC operations, and ends with simulations, proof of skill, certification, and client-readiness preparation.

## Non-negotiables

- Existing ProjectAmazonPH repositories remain untouched.
- No AI features or LLM dependencies.
- Test-driven development is required.
- Iteration continues until documented release gates pass.
- Mobile accessibility, honest claims, and simulated-work disclosure are mandatory.
