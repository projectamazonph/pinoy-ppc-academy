# PPC Foundations Track

**Product:** Pinoy PPC Academy  
**Repository:** `pinoy-ppc-academy`  
**Version:** 1.0  
**Status:** Documentation baseline

## Purpose

Introduces Amazon advertising concepts, campaign types, targeting, and metrics.

## Track outcome

The learner understands Sponsored Products, Sponsored Brands, and Sponsored Display; can calculate core metrics; can distinguish targets from shopper search terms; and can make safe basic decisions.


## Modules

| Module | Key content |
| --- | --- |
| Advertising Economics | CPC, CTR, CVR, ACoS, ROAS, TACoS, break-even ACoS |
| Campaign Types | SP, SB, SD, use cases, limitations, creative requirements |
| Targeting and Match | Automatic targeting, exact, phrase, broad, ASIN, category, audience |
| Bids and Placements | Default bids, target bids, placement multipliers, bidding strategies |
| Budgets and Pacing | Daily budgets, caps, budget rules, pacing, wasted opportunity |
| Search Term Logic | Keyword versus search term, harvesting, negative exact, negative phrase |


## TACoS requirement

TACoS receives a dedicated lesson and calculator. Students compare accounts with identical ACoS but different TACoS and explain what each movement says about organic sales, advertising dependence, and business health.


## Foundation gate

- [ ] At least 80 percent on the knowledge assessment.
- [ ] Correct metric calculations with units and interpretation.
- [ ] Successful guided campaign build.
- [ ] Successful search-term classification exercise.
