# VA Career Starter Track

**Product:** Pinoy PPC Academy  
**Repository:** `pinoy-ppc-academy`  
**Version:** 1.0  
**Status:** Documentation baseline

## Purpose

The free entry path for learners with no VA experience or those changing careers.

## Track outcome

The learner understands how virtual assistant work operates, can prepare a safe and credible work setup, identifies transferable skills, avoids common scams, and chooses Amazon PPC with realistic expectations.


## Modules

| Module | Topics | Evidence |
| --- | --- | --- |
| 1. VA Work Reality | Remote work models, responsibilities, time zones, contracts, boundaries | Work-readiness checklist |
| 2. Home Work Setup | Internet, backup plans, device requirements, file organization, security | Personal setup plan |
| 3. Core Digital Skills | Email, calendars, cloud files, spreadsheets, task systems, screenshots | Tool skills assessment |
| 4. Communication | Clear updates, clarifying questions, escalation, meeting notes, professional English | Client update exercise |
| 5. Career Shift Mapping | Transferable skills, experience stories, gap analysis, learning plan | Career shift map |
| 6. Client Safety | Scam signals, payment safety, confidentiality, trial tasks, account access | Risk scenario score |
| 7. Amazon Career Map | Amazon business roles and why advertising is a high-value specialization | Specialization decision |
| 8. Readiness Assessment | Knowledge and setup review | Recommended learning route |


## Free path boundaries

- The free track is complete enough to be useful without payment.
- It does not include full PPC simulators or certification.
- Upgrade prompts appear only after relevant lessons or readiness results.
- The student can download their career shift map and setup plan.


## Acceptance criteria

- [ ] A learner with no Amazon knowledge can finish without undefined jargon.
- [ ] Every module works on a 360px-wide viewport.
- [ ] The readiness route is deterministic and covered by tests.
- [ ] No lesson promises employment or a specific income.
