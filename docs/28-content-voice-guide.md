# Content and Voice Guide

**Product:** Pinoy PPC Academy  
**Repository:** `pinoy-ppc-academy`  
**Version:** 1.0  
**Status:** Documentation baseline

## Purpose

Defines how the product speaks to Filipino career starters and PPC learners.

## Voice

Direct, patient, practical, and respectful. The platform should sound like an experienced PPC lead teaching at a whiteboard after work, not a motivational speaker and not a textbook.


## Core rules

- Define every acronym the first time it appears.
- Use real numbers and show the calculation.
- Use Philippine context naturally, not as decoration.
- Address the learner directly.
- Explain what happened, why it happened, and what to do next.
- Separate verified fact from recommendation or scenario assumption.
- Never promise income, employment, or guaranteed advertising results.


## Examples

| Avoid | Use |
| --- | --- |
| Unlock your potential with comprehensive training. | Learn the PPC tasks agencies expect you to perform. |
| Invalid input. | Enter a daily budget above PHP 0. |
| Leverage advanced optimization strategies. | Use search-term data to decide what to harvest, negate, or leave alone. |
| Your journey starts here. | Start with the free VA Career Starter path. |
| Incorrect. | This negative phrase would also block a converting search term. Use negative exact instead. |


## Filipino context rules

- Use pesos for course pricing and locally relevant examples.
- Use city names only when they add real context.
- Do not force Tagalog into English UI labels.
- Do not use "po" or "opo" as generic product voice.
- Acknowledge mobile learning, night-shift schedules, and backup internet realities without stereotyping.
