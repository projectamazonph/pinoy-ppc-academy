# PPC Strategist Track

**Product:** Pinoy PPC Academy  
**Repository:** `pinoy-ppc-academy`  
**Version:** 1.0  
**Status:** Documentation baseline

## Purpose

Builds diagnosis, account planning, scaling, and commercial reasoning.

## Track outcome

The learner can move beyond isolated bid changes, identify root causes, create a strategy tied to business goals, prioritize work, and explain tradeoffs to a client.


## Modules

| Module | Strategic capability |
| --- | --- |
| Diagnostic Framework | Classify problems before acting: bid, listing, inventory or price, structure |
| Portfolio Strategy | Allocate budgets by defense, research, performance, launch, and testing roles |
| Search Term Systems | Harvesting thresholds, isolation, negative safety, query overlap |
| Bid Economics | Target CPC, CVR-based bids, break-even logic, placement economics |
| Scaling | Increase spend while protecting contribution margin and inventory |
| Turnarounds | Stabilize high ACoS, broken structures, suppressed listings, and budget constraints |
| Seasonality | Promotions, Q4, events, inventory lead time, pacing changes |
| Business Reporting | Explain what moved, why, what was done, and what happens next |


## Decision standard

Every strategy recommendation must include the observed signal, diagnosis, action, guardrail, expected outcome, and review window. "Increase bids" is not a strategy.


## Strategist gate

- [ ] Complete an account audit with prioritized findings.
- [ ] Produce a 30-day strategy with budgets, campaign changes, and measurement plan.
- [ ] Defend three decisions under questioning.
- [ ] Identify at least one case where the correct action is to leave the campaign unchanged.
